# scraper-uji
Este proyecto implementa un scraper automatizado en Python para recopilar noticias publicadas en la web institucional de la Universitat Jaume I (UJI). El script navega por el buscador de noticias, accede a cada entrada individual y descarga su contenido en tres formatos: HTML, texto plano y Markdown, generando además un índice JSON estructurado.
